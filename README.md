# LS45 Automotriz

## Descripción del proyecto

Sitio web desarrollado para LS45 Automotriz utilizando HTML, CSS, Bootstrap y JavaScript, basado en la adaptación de una plantilla web automotriz.

## Objetivo

Personalizar un template web incorporando contenido relacionado con una empresa automotriz, mejorando la navegación, el diseño y la experiencia de usuario.

## Secciones incorporadas

- Inicio
- Nosotros
- Servicios
- Vehículos
- Contacto

## Funcionalidades implementadas

- Navegación interna mediante identificadores ID
- Diseño responsive para dispositivos móviles
- Enlaces funcionales para correo, teléfono y ubicación
- Integración de redes sociales
- Adaptación de imágenes, colores y textos
- Eliminación de elementos innecesarios del template original

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap
- JavaScript
